# Gemini-for-zev
🚀 Gemini 助手 (Gemini Assistant) 功能简介
这是一个专为 Google Gemini 打造的全方位效率增强脚本，集成了回答导航、界面美化与实用工具，显著提升长对话的阅读与管理体验。

核心功能：

📍 精准回答跳转与标记

在每个回答下方新增图钉按钮（📌），支持一键标记重点回答。

提供侧边悬浮导航球，点击展开面板，可快速跳转至已标记的回答位置。

智能状态反馈：未标记显示实心图标，已标记显示描边图标；悬浮球会根据是否有标记内容自动切换颜色与状态。

🎨 现代简约 UI 设计

极致美化：采用磨砂玻璃特效（Backdrop Blur）面板与 Google Material 风格圆角设计。

流畅交互：支持点击面板外部空白区域自动收起，全系采用美化后的确认弹窗，告别原生丑陋提示。

定制图标：内置高清 SVG 图标库，视觉清晰锐利。

🛠️ 完美布局修复

智能插入：精准定位 Gemini 复杂的 DOM 结构，将功能按钮无缝嵌入到原生按钮栏的最左侧（点赞按钮前）。

拒绝对齐错乱：修复了按钮挤压、换行或距离过远的问题，与原生界面浑然一体。

🖥️ 阅读体验增强

宽屏模式：支持一键开启宽屏（1800px），充分利用大屏幕空间。

多主题切换：内置护眼模式、深色/浅色模式等多种配色方案。

📂 实用工具箱

Markdown 导出：支持将当前对话完整导出为 Markdown 格式，保留用户与 AI 的角色区分。

一键到底：提供悬浮按钮，快速滚动至对话最底部。
