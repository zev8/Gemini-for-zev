# Gemini 助手 (Gemini Assistant)

一个功能强大的 Google Gemini 油猴 (Tampermonkey) 增强脚本，专注于提升阅读体验、导航效率和界面美观度。

优化护眼模式

* **ps:使用时请切换主题到浅色模式**

## ✨ 主要功能

* **🚀 高效导航**
    * 提供悬浮按钮，一键跳转到 **上一个/下一个提问**（自动顶部对齐）。
    * 一键 **滚动到底部**。
    * *智能防误触*：精准定位主对话窗口，避免误滚动左侧历史栏。

* **📌 回答锚点 (书签)**
    * 在每个回答右上角添加 **图钉图标**，点击即可标记该回答。
    * 提供侧边悬浮面板，列出所有已标记的回答，点击即可快速跳转。
    * 支持对锚点进行 **重命名** 和 **删除**。

* **📂 智能导出**
    * 支持将当前对话导出为 **Markdown (.md)** 文件。
    * **智能命名**：自动抓取当前对话标题作为文件名（如未生成标题则使用网页标题）。

* **🎨 界面与主题增强**
    * **极简线性图标**：全套重绘的 SVG 线性图标，风格统一、圆润高级。
    * **多主题切换**：内置 浅色、护眼、舒适、深色 4 种显示模式。
    * **宽屏模式**：一键开启 1800px 宽屏显示，利用大屏优势。

## 🛠️ 安装方法

1.  安装浏览器扩展程序 **Tampermonkey** (油猴)。
2.  点击 Tampermonkey 图标 -> "添加新脚本"。
3.  将 `gemini_assistant_v8.9.js` 中的代码完整复制并粘贴到编辑器中。
4.  按 `Ctrl + S` 保存即可。

## 📖 使用说明

* **功能面板**：位于页面右下角，包含主题切换、宽屏开关和导出按钮。
* **导航按钮**：位于页面右下侧边缘，包含上/下跳转和到底部按钮。
* **锚点管理**：
    * 鼠标悬停在任意回答上，点击出现的 **图钉图标** 进行标记。
    * 点击屏幕右侧中部的 **悬浮球** 打开锚点列表面板。
